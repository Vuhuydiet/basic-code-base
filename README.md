# React Tic Tac Toe App

A simple counter application built with React, TypeScript, Context API, and Tailwind CSS.

## How to Run

### Prerequisites

- Node.js (version 16+)
- npm

### Installation & Run

1. **Install dependencies:**

```bash
npm install
```

2. **Start development server:**

```bash
npm run dev
```

3. **Open browser:**
   Go to `http://localhost:5173`
